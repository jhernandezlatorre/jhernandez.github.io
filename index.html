Página personal de Jorge Hernández

Esta es mi página personal fruto de la confección de un trabajo realizado para completar el segundo módulo del Mooc “Desarrollo en HTML5, CSS y JavaScript de Apps Web”.

Dentro de éste Mooc entiendo que vamos a aprender el lenguaje de marcas HTML5, las hojas de estilos CSS3 y el lenguaje Javascript.

HTML5

Francamente reconozco que hasta ahora no lo conocía bien, he trabajado en el pasado con versiones anteriores y tengo mucha ilusión en tratar de dominar esta versión 5 del mismo. La idea que tengo de las anteriores versiones es que lo principal es muy fácil pero en cuanto las necesidades de la aplicación son algo elevadas resulta muy “engorroso” el tratar de situar toda la información en la pantalla con cierta gracia sin que se desajusten las diferentes cajas de texto. Con la orden “float” de las cajas realizadas con <div><div> no hay quien se entienda si entramos en situaciones exigentes. El problema se agravaba cuando se tenía que desarrollar para para distintos navegadores donde uno se veía abocado a meter pequeños “scripts” en javascript que al final lo complican todo mucho. Espero con esta versión 5 que todo los recuerdos pasados queden en eso recuerdos de ciertos tormentos que no volveremos a sufrir.

CSS3

Por lo que estoy ojeando es muy reconfortante observar que se ha avanzado mucho en la presentación del contenido. Hay instrucciones que con mucha simplicidad permiten crear efectos muy profesionales sin acudir a librerías como se hacía en las versiones anteriores.

Javascript

Es la parte de largo que más me interesa del Mooc ya que sigo de cerca la evolución de los diferentes lenguajes de computación y veo que claramente Javascript no solamente es un sobreviviente sino que se ha impuesto a muchos de los que se podían intuir como insustituibles. Al principio de los tiempos HTML, Javascript se limitaba simplemente a pequeños script que corrían exclusivamente en el equipo cliente dentro de un navegador y que era empleado para validar datos o para crear algún que otro efecto visual con aspiración a parecer profesional. Ahora mismo la consideración ha cambiado del todo, Javascript no solo sigue siendo la estrella dentro de los navegadores siendo el centro de los web sites, sino que también ha transgredido a la parte del servidor resultando imprescindible para cualquier seguidor de la tecnología moderna su conocimiento.

  2. Estructura HTML5/CSS3 para multipantalla y objetos multimedia

  5. Bucles. Arrays. Funciones como objetos. Ámbitos. Cierres. jQuery
  6. jQuery UI. Interacción con usuario. Bootstrap. Diseño adaptativo
  7. Profundización en arrays, objetos, propiedades, métodos, tipos
  8. LocalStorage, iFrames, Origin policy y JSON
  9. Introducción a gráficos y animaciones SVG. Geolocalización.
  10. Empaquetar aplicaciones Web para Android o iOS
